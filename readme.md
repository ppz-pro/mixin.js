# mixin.js
给 js 增加 mixin 特性

mixin 自然是 OO（Object Oriented，面向对象）的特性  
而 OO 近年的名声不是很好，特别是 js 的 OO  
但我喜欢 OO，也相信还有更多人喜欢 OO 编程

js 在 OO 方面的坏名声，并不是因为功能少  
而是“先天设计畸形”  
所以这里增加 mixin 特性，并不为挽回 js 的名声  
js 在 OO 方面的表现确实糟糕  

mixin.js 的意义，仅在于增加这么一个特性

``` bash
npm install @ppzp/mixin
```
